### Что это такое?
 - Полное название данной ОС: `ClemOS`
 - Разрядность: `x86`
 - Ядро: `Модульное`
 - Ядро написано на языке `NASM`.
____
 
### Сборка.
 Для сборки вам потребуется скачать `NASM` с официального сайта nasm.us
* Затем,
  * `Win32`:
   * Запустить `build.bat`
  * `Linux`:
   * Извините, но я временно не пользуюсь им...
